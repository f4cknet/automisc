## 3. 详细设计

### 3.1 `_extract_lsb_byte_stream` 修复 (per-pixel interleaved)

**目标**: 跟 zsteg `b1,rgb,lsb,xy` 完全等价

**新实现**：
```python
def _extract_lsb_byte_stream(arr, channels, bit, scan_order, byte_bit_order):
    """zsteg-compatible per-pixel interleaved LSB byte stream."""
    ch_indices = [_channel_index(c) for c in channels]
    n_ch = len(ch_indices)
    if scan_order == "row":
        # (H, W, n_ch) interleaved, flatten row-major → per-pixel: R0 G0 B0 R1 G1 B1 ...
        plane = arr[:, :, ch_indices]                    # (H, W, n_ch)
        flat = plane.reshape(-1, n_ch)                   # (H*W, n_ch) row-major
    else:  # col
        # transpose → column-major flatten → per-pixel: R0 G0 B0 R1 G1 B1 ...
        plane = arr[:, :, ch_indices]                    # (H, W, n_ch)
        plane_t = plane.transpose(1, 0, 2)               # (W, H, n_ch)
        flat = plane_t.reshape(-1, n_ch)                 # (W*H, n_ch) col-major
    bits = ((flat >> bit) & 1).astype(np.uint8).flatten(order='C')  # 默认 C-order = row
    n_bytes = len(bits) // 8
    bits_trimmed = bits[: n_bytes * 8]
    if byte_bit_order == "msb":
        return np.packbits(bits_trimmed).tobytes()
    # lsb: 字节内 bit 反序
    bits_2d = bits_trimmed.reshape(-1, 8)[:, ::-1].flatten()
    return np.packbits(bits_2d).tobytes()
```

### 3.2 `_build_bit_plane_preview_matrix` 新建

**目标**: 8 bit × 6 perm preview matrix, ASCII 可读

**新实现**：
```python
def _build_bit_plane_preview_matrix(arr: np.ndarray, n_bytes: int = 32) -> dict[tuple[int, str], str]:
    """生成 8 bit × 6 perm preview 矩阵 (Owner 要求 "超过随波逐流").

    Args:
        arr: (H, W, 3) RGB numpy array
        n_bytes: 每个 combo 前 N 字节 preview (默认 32)

    Returns:
        dict[(bit, perm_name), ascii_preview_str]
    """
    PERMS = [
        ("RGB", [0, 1, 2]),
        ("RBG", [0, 2, 1]),
        ("GRB", [1, 0, 2]),
        ("GBR", [1, 2, 0]),
        ("BRG", [2, 0, 1]),
        ("BGR", [2, 1, 0]),
    ]
    matrix = {}
    for bit in range(8):
        for perm_name, ch_idx in PERMS:
            # 调用修复后的 _extract_lsb_byte_stream (per-pixel)
            byte_stream = _extract_lsb_byte_stream(
                arr, channels=[_idx_to_channel(i) for i in ch_idx],
                bit=bit, scan_order="row", byte_bit_order="msb",
            )
            # ASCII preview, 控制字符 → '.'
            preview = "".join(
                chr(b) if 32 <= b < 127 else "."
                for b in byte_stream[:n_bytes]
            )
            matrix[(bit, perm_name)] = preview
    return matrix
```
