## v0.5 release 标准

- 总 adapter ≥ 30（v0.1 = 22 + v0.5 = +8 adapter）
- 测试 ≥ 350（v0.1 = 276 + v0.5 = +50~80 新测试）
- 真实 fixture 全套（vmem + evtx + 大 pcap + 大 wav + 4K png）
- 文档完整（upgrade.md 索引 ≥ 8 个迭代）
- GUI 截图（4 屏：drop PNG / drop PCAP / 菜单手动 / journal 满载）

---

## v0.5 release 后 → v1.0

v1.0 范围（per `prd.md §9`）：
- 跨平台（Windows + Linux）— 但 v0.1 约束 macOS only；v1.0 突破
- 22+ adapter 完整 + Office 套件
- GUI 高保真（dark mode / 全键盘操作 / 拖拽多文件）
- Web 视图（轻量 HTTP server + 浏览器控制）

## 决策记录
