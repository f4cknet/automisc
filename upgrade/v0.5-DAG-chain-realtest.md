## 完整攻击链（CLI 验证）

### Step 0: 初步分析

```bash
$ file Challenge/QR_code.png
PNG image data, 280 x 280, 1-bit colormap, non-interlaced

$ automisc run --tool binwalk --file Challenge/QR_code.png
=== binwalk ===
exit_code: 0
suspicious_points (2):
  [4] file_header: PNG image @ offset 0
  [4] file_header: ZIP archive @ offset 471
```

**关键发现**：binwalk 检测到 PNG @ 0 + **ZIP @ offset 471**（内嵌 zip）

### Step 1: binwalk 链（自动提取）

```bash
$ automisc chain --chain binwalk --file Challenge/QR_code.png
[1] binwalk_extract    OK    foremost extracted 2 files
  -> /tmp/.../foremost_out/png/00000000.png  (原图)
  -> /tmp/.../foremost_out/zip/00000000.zip  (内嵌 zip!)
```

### Step 2: zip 链分析（真伪加密识别）

```bash
$ automisc chain --chain zip --file .../foremost_out/zip/00000000.zip
[1] try_unzip          FAIL  zip is encrypted
[2] fix_pseudo_encryption FAIL  fix failed, restored from backup: invalid block
```

- ✅ 正确识别**真加密**（fix_pseudo 失败 = 不是伪加密）
- 自动还原 from backup

### Step 3: 完整爆破链

```bash
$ automisc chain --chain zip-full --file .../00000000.zip --bruteforce-limit 15000
[1] try_unzip          FAIL  zip is encrypted
[2] fix_pseudo_encryption FAIL  fix failed
[3] bruteforce_zip     OK    FOUND password='7639' (tried 7640/15000)
```

**爆破成功**：4 位数字密码 `7639` 在 7640 次尝试内找到

### Step 4: 读 flag

```bash
$ cat .../foremost_out/zip/00000000_bruteforced/4number.txt
CTF{vjpw_wnoei}
```

## 关键技术收获

1. **binwalk -e 在 macOS 不可靠** — 需要 `dd` 工具，silently 失败
2. **foremost 是 macOS 提取的真可靠工具** — v0.5-DAG-chain 改 binwalk 检测 + foremost 提取
3. **真伪加密识别准确** — `_is_pseudo_encrypted` 用 raw bytes 检测 12 字节 PKCS#5 header，正确区分
4. **4 位数字字典爆破有效** — 8.4M 全字典太慢，`--bruteforce-limit 15000` 就能找到弱密码
5. **链式编排** — try → fix → bruteforce 顺序正确，失败转移无误

## 改进（基于实测）

| 项 | 改前 | 改后 |
|---|---|---|
| `BinwalkExtractAction` | 用 `binwalk -e` 提取 | `binwalk` 检测 + `foremost -t all` 提取（macOS 可靠）|
| `_is_pseudo_encrypted` | 用 `python zipfile.read(pwd=...)` 试解压 | 读 raw bytes + 12 字节 PKCS#5 header 检测 |
| BruteforceZipAction | 全字典 8.4M | context 接受 `__bruteforce_limit__` 限制（v0.1 简化）|
| CLI `chain` 子命令 | 无 | 新增 `automisc chain --chain {zip,zip-full,binwalk} --file` + `--bruteforce-limit` + `--keep-backup` |
| `cmd_chain` 输出 | N/A | router 推荐 + chain log + summary + last action data + 自动 cleanup bak |
