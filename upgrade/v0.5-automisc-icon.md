## 3. 实施清单

### 3.1 main_gui 入口挂图标

`src/automisc/__main__.py:main_gui`（line 63-79）加：

```python
from PySide6.QtGui import QIcon
from pathlib import Path

# 解析图标路径 — 兼容开发模式 (src/automisc.ico) 和未来 PyInstaller 打包
_ICON_CANDIDATES = [
    Path(__file__).parent / "automisc.ico",            # 优先: 跟 __main__.py 同目录
    Path(__file__).parent.parent / "automisc.ico",     # fallback: src/automisc.ico
    Path(__file__).parent.parent.parent / "automisc.ico",  # fallback: repo 根
]

def _find_icon() -> Path | None:
    for p in _ICON_CANDIDATES:
        if p.exists():
            return p
    return None

# 在 main_gui 里:
app = QApplication.instance() or QApplication(sys.argv)
icon = _find_icon()
if icon is not None:
    app.setWindowIcon(QIcon(str(icon)))
```

`MainWindow` 加 fallback（标题栏左上角）：

```python
# src/automisc/gui/main_window.py MainWindow.__init__
icon_path = _find_icon()  # 复用上面 helper, 提到共享位置
if icon_path is not None:
    self.setWindowIcon(QIcon(str(icon_path)))
```

> **Why both**：`app.setWindowIcon()` 给所有 QWidget 默认；`MainWindow.setWindowIcon()` 显式 override 标题栏左上角. 两个都设保险.
