Owner 实测后问："既然python-evtx能代替 evtx-dump，那干脆统一用python-evtx不就行了吗？"

**核心洞察**：当前 adapter `src/automisc/tools/forensics/log/evtx_dump.py` 实际**完全用 python-evtx**（`import Evtx.Evtx as evtx_mod`），不调任何外部 binary。CLI 价值 = Owner 手动 `evtx_dump file.evtx | grep flag` 的便利，可被 5 行 Python one-liner 替代。

### 1.3 当前 adapter 代码（216 行 · per `src/automisc/tools/forensics/log/evtx_dump.py`）

```python
# 第 75 行
import Evtx.Evtx as evtx_mod
# 第 89-100 行
with evtx_mod.Evtx(file_path) as log:
    for record in log.records():
        xml_str = record.xml()  # Python 直接拿 record 对象
```

**6 类 domain logic 全用 python-evtx in-process**：

| 维度 | 实现 |
|---|---|
| EventID 4625 / 1102 等可疑事件打分 | XPath `e:System/e:EventID` + `_SUSPICIOUS_EVENTIDS` 映射 |
| 4688 进程命令行字段 | `e:EventData/e:Data[@Name='CommandLine']` |
| Severity 动态升级（命中 `mimikatz` `-enc` → sev+1）| 字段级精准匹配 |
| 错误处理 | `try/except` 直接捕获 |
| 5000 records 上限 + OOM 保护 | Python 内存可控 |
| 单 adapter call 启动开销 | ~0（in-process）|

**结论**：adapter 路径上 evtx_dump CLI **0 调用**。

## 3. 已实施的改动（commit `f54d859`）

### 3.1 install.ps1 — Stage 0 Rust 装

```powershell
# Stage 0: Rust toolchain (optional, per v0.5-windows-evtx-dump)
$rust_installed = $false
try {
    $ver = & rustc --version 2>$null
    if ($LASTEXITCODE -eq 0) {
        Write-Host "[skip] rustc already installed: $ver"
        $rust_installed = $true
    }
} catch {}

if (-not $rust_installed) {
    try {
        Write-Host "[download] rustup-init.exe ..."
        $rustup = Join-Path $StageDir "rustup-init.exe"
        Download-File -Url "https://win.rustup.rs/x86_64" -Out $rustup
        Write-Host "[install] rustup-init -y --default-toolchain stable --profile minimal ..."
        & $rustup -y --default-toolchain stable --profile minimal | Out-Null
        if ($LASTEXITCODE -ne 0) { throw "rustup-init exit $LASTEXITCODE" }
        # Refresh PATH
        $env:Path = [System.Environment]::GetEnvironmentVariable('Path','Machine') + ';' + [System.Environment]::GetEnvironmentVariable('Path','User')
    } catch {
        Write-Host "warning: $_ (Rust not installed; install.ps1 will continue)"
    }
}
```

**特点**：
- idempotent（rustc 已装 → skip）
- 失败 warning continue（不阻塞 Stage 1 / Stage 2）
- minimal profile 跳过 docs 省 ~200MB
- 装完 refresh PATH 让后续 Stage 立刻能用 cargo
