**Owner 提示**：跟 [随波逐流] CTF 编码工具对比，随波逐流一眼看到 LSB RGB 通道明文。Owner 要求 automisc "LSB + MSB 都打印，比随波逐流更强"。

#### 2.2.1 手动 Python 验证 per-pixel interleaved (zsteg 标准)

```python
import numpy as np
from PIL import Image
img = Image.open(steg_path).convert("RGB")
arr = np.array(img)
# per-pixel: R0 G0 B0 R1 G1 B1 ... (zsteg `b1,rgb,lsb,xy` 标准)
flat = arr.reshape(-1, 3)  # (h*w, 3)
bits = ((flat >> 0) & 1).flatten()
packed = np.packbits(bits[:8000]).tobytes()  # MSB first
# → 前 64 字节 ascii: "Hey I think we can write safely in this file witho"
```

**结论**: zsteg 标准 per-pixel interleaved **能**完整读出明文。

#### 2.2.2 automisc `_extract_lsb_byte_stream` 实际输出（plane-separated bug）

```python
# automisc 当前实现 (lsb_tool_common.py:155-170):
for ch in channels:                       # ['R','G','B']
    channel_data = img_array[:, :, idx]   # 整个 R 通道 (HxW)
    bits = ((channel_data >> bit) & 1).astype(np.uint8).flatten()
    bits_list.append(bits)
all_bits = np.concatenate(bits_list)      # 整 R + 整 G + 整 B
```

**对比**：

| | automisc（plane-separated）| 正确（per-pixel interleaved）|
|---|---|---|
| 前 8 字节 hex | `16 1e 5a 3e 76 37 16 13` | `48 65 79 20 49 20 74 68` |
| ASCII | `..Z>v7..` | `Hey I th` |

**根因**: v0.5-lsb-tool-unify 合并 3 个 LSB action 时，`lsb_detect.py:367` 已经注释说 "per pixel 3 通道交错顺序, **跟 zsteg 一致**, 之前 d66177c commit 改 per channel 连续是错的"，但 v0.5-lsb-tool-unify 把这段逻辑抽到 `lsb_tool_common.py:_extract_lsb_byte_stream` 时**实现又写回 plane-separated**，bug 回归。

## 4. automisc 应补能力（架构升级触发）

### 4.1 🚨 Bug: `_extract_lsb_byte_stream` plane-separated 实现错（架构升级触发）

**根因**: `src/automisc/core/actions/lsb_tool_common.py:155-170` 把 6 个 perm 全按 plane-separated 实现，**所有 6 个 perm 字节流全错位**，auto-run 永远命中不了明文。

**修复**: 改 per-pixel interleaved（zsteg 标准）：
```python
# 正确做法
interleaved = img_array.reshape(-1, len(channels))[:, channels_idx]
bits = ((interleaved >> bit) & 1).flatten()  # R0 G0 B0 R1 G1 B1 ...
```

**架构判定**:
- 这是 v0.5-train-011（lsb_detect HWC 修复）+ v0.5-train-012（字节流同源）的**bug 回归**
- 同一 bug 多次实战触发（≥3 道同类：v0.5-train-011/012/014）→ 架构升级 ✅

**触发**: Owner 2026-06-28 12:33 拍板"LSB+MSB 都打印，比随波逐流更强"
