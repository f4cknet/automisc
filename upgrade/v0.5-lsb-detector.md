## 1. 背景

### 1.1 问题陈述 (per v0.5-train-010 §2-§3)

N=NP (大图副本, 1253×834 RGBA PNG, 1.6MB) auto-run 实战:
- **zsteg 单 G 通道 LSB 输出空** — zsteg 默认只显示"匹配到 printable text" 的组合, 单 G 通道 LSB 字节流不可读 = 显示空
- **zsteg 不试 col 扫描** (固定 xy=row) — 写死
- **zsteg 不区分"真可疑 text" vs "假命中 printable bytes"** — 3 条 lsb_text 全 severity=4 (info), 没办法分级
- **zsteg 不出"单通道异常"概率提示** — N=NP 模式 G 通道完整 8 bit 隐藏 pyc, zsteg 完全没有这个能力

AI 兜底验证 (per §3.2):
- **G 通道 LSB (b0) entropy=0.9854** (max=1.0) — 比 R 高 4.6%, 比 B 高 12.8% — 异常信号
- **G 通道字节流 entropy=5.6792**, unique=**256/256 (100%) 满** — R/B 缺 40+ 字节
- 1.5 σ 异常, **真信号** — 假设立竿见影

## 2. 目标

### 2.1 能力 A: 3 通道 LSB (bit 0) 智能检测 (替代 zsteg)

**目标**: 对 RGB 3 通道, bit 0 (LSB 最低位), 6 排列 (RGB/RBG/GRB/GBR/BRG/BGR) × 2 scan (row/col) = **12 组合**, 每组合抽字节流, 智能判定 text / 文件头 / 跳过。

**判定逻辑** (per Owner Q3=A 拍板):
1. **text 判定** (per Owner 21:29 + 22:00 拍板):
   - 字节流全 printable ASCII (32-126 区间) = text
   - **不**满足 = 不是 text, 走文件头判定
   - 命中 → SP severity=5 cat=lsb_text
2. **文件头判定** (per Owner Q3=A 拍板, **双机制**):
   - **hex magic 主判定**: 前 N 字节匹配 known magic signature (复用 v0.5-lsb-byte-stream-extract `magic_sniffer` 50+ entry 库)
   - **`file` 命令辅判定**: libmagic 兜底
   - 命中 → SP severity=5 cat=lsb_file_header (sub-cat=zip/rar/png/pyc/elf/docx/...)
3. **都不是** = 跳过 (不写 SP, 避免噪声)

**输出格式** (per Owner 例子):
- text 命中: `"lsb_detect 发现 lsb RGB row_first 存在可疑内容: AfKg^pL :"` sev=5 cat=lsb_text matched_pattern=截断 200 字符
- 文件命中: `"lsb_detect 发现 lsb BGR col_first 存在可疑 pyc 文件: 03 f3 0d 0a 62 0d ..."` sev=5 cat=lsb_file_header sub_cat=pyc matched_pattern=截断 200 字符

### 2.2 能力 B: 单通道 8 bit 隐藏信息概率检测

**目标**: 对 R/G/B 3 通道, 各自**完整 8 bit** 拼成字节流, 算 entropy + unique count, 跨通道比较找异常。

**算法** (per Owner Q4=A 拍板):
1. PIL 读图 → numpy 拆 R/G/B
2. 每通道 8 bit plane → 拼 byte 字节流 (8 个 bit plane 当字节位, 详见 v0.5-train-010 §3.2)
3. 算 2 个指标: **entropy** (香农熵, max=8.0) + **unique byte count** / 256
4. 跨通道比较: 找 entropy 异常高 (per channel entropy > 5.0) + unique 满 (>= 250/256)
5. 写 SP severity=4 (info, **不**像需求 1 是 severity=5) — per Owner "**没有绝对性**"

**MVP 阈值** (per v0.5-train-010 §3.2 验证数据):
- entropy > 5.0 (max=8.0) = 单通道异常
- unique >= 250/256 (97.7%+) = 字节值覆盖广
- 跨通道 entropy 差 > 0.5 (绝对差) = 突出

**输出格式** (per Owner 例子):
- 单通道异常: `"lsb_detect 发现 Green plane 存在可疑 pyc 文件"` sev=4 cat=lsb_channel_anomaly
- matched_pattern: `entropy=5.68 unique=256/256 (R=4.18/B=4.21, 突出 1.5 σ)`

### 3.2 核心算法 (LSBDetectAction.run)

```python
def run(self, file_path: str) -> list[SuspiciousPoint]:
    """3 通道 LSB bit 0 智能检测 + 单通道 8 bit 概率检测.
    
    Returns:
        list[SuspiciousPoint]: 命中 SP (text/file_header severity=5, channel_anomaly severity=4)
    """
    from PIL import Image
    import numpy as np
    
    img = Image.open(file_path).convert('RGB')  # Q2=A RGB 3 通道
    arr = np.array(img)  # shape (H, W, 3)
    
    sps = []
    
    # ========== 需求 1: 3 通道 LSB (bit 0) 智能检测 ==========
    permutations = [
        (0, 1, 2), (0, 2, 1), (1, 0, 2), (1, 2, 0), (2, 0, 1), (2, 1, 0),  # RGB 全排列 6
    ]
    scan_orders = ['row', 'col']
    
    for perm in permutations:  # 6 排列
        for scan in scan_orders:  # 2 scan
            # 抽字节流
            bit0_planes = arr[:, :, perm] & 1  # shape (H, W, 3) 各通道 bit 0
            if scan == 'row':
                bits_flat = bit0_planes.flatten()  # row first
            else:  # 'col'
                bits_flat = bit0_planes.T.flatten()  # col first
            # 8 bit 拼 byte
            n_bytes = len(bits_flat) // 8
            byte_stream = np.zeros(n_bytes, dtype=np.uint8)
            for bit_pos in range(8):
                byte_stream |= (bits_flat[bit_pos::8][:n_bytes] << bit_pos).astype(np.uint8)
            
            # 判定 1: text (printable ASCII 32-126)
            if is_printable_text(byte_stream):
                sps.append(SuspiciousPoint(
                    tool_name="lsb_detect",
                    category="lsb_text",
                    matched_pattern=f"lsb_detect 发现 lsb {perm_name(perm)} {scan} 存在可疑内容: "
                                   f"{bytes_preview(byte_stream, 200)}",
                    severity=5,  # 真可疑
                ))
                continue  # 命中 text, 不再走文件头判定
            
            # 判定 2: 文件头 (hex magic + file 双机制)
            magic_hit = detect_file_header(byte_stream)  # 复用 magic_sniffer
            if magic_hit:
                sps.append(SuspiciousPoint(
                    tool_name="lsb_detect",
                    category="lsb_file_header",
                    matched_pattern=f"lsb_detect 发现 lsb {perm_name(perm)} {scan} 存在可疑 {magic_hit.ext} 文件: "
                                   f"{bytes_preview(byte_stream, 200)}",
                    severity=5,  # 真可疑
                    context={"file_type": magic_hit.ext, "magic": magic_hit.desc},
                ))
                # continue  # 不 continue, 允许 text + file 都报 (罕见但可能)
    
    # ========== 需求 2: 单通道 8 bit 概率检测 ==========
    for ch_idx, ch_name in enumerate(['R', 'G', 'B']):
        # 8 bit 拼 byte 字节流
        plane = arr[:, :, ch_idx]
        bits_per_plane = [(plane >> bit) & 1 for bit in range(8)]
        # 8 个 bit plane 拼 byte: 每 8 个像素同一 bit 拼一个字节
        n_bytes = plane.size // 8
        byte_stream = np.zeros(n_bytes, dtype=np.uint8)
        for bit_pos in range(8):
            byte_stream |= (bits_per_plane[bit_pos].flatten()[:n_bytes * 8][::8][:n_bytes] << bit_pos).astype(np.uint8)
        
        ent = shannon_entropy(byte_stream)
        unique = len(np.unique(byte_stream))
        
        # 阈值
        if ent > 5.0 and unique >= 250:
            # 跨通道 entropy 差
            all_ents = [shannon_entropy(channel_8bit_byte_stream(arr[:, :, i])) for i in range(3)]
            max_diff = max(all_ents) - min(all_ents)
            
            sps.append(SuspiciousPoint(
                tool_name="lsb_detect",
                category="lsb_channel_anomaly",
                matched_pattern=f"lsb_detect 发现 {ch_name} plane 存在可疑隐藏信息 "
                               f"(entropy={ent:.2f} unique={unique}/256, 跨通道差 {max_diff:.2f})",
                severity=4,  # info, 概率 (per Owner "没有绝对性")
                context={"channel": ch_name, "entropy": ent, "unique": unique, "max_diff": max_diff},
            ))
    
    return sps
```
