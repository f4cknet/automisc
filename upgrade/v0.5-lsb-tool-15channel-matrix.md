## 1. 背景

### 1.1 UX Gap: lsb_tool vs 随波逐流 视图对比

**Owner 2026-06-28 17:18 实战反馈**:
> "lsb_tool auto OK 但没出 SP, 切到 随波逐流 后 RGB 通道直接出明文 `Hey! I think we can write safely in this file witho...`"

**Owner 2026-06-28 18:06 实战截图反馈** (v2 触发):
> "好像没修好（左图），我希望的效果（右图）"
> 左图: 8 bit × 6 perm matrix 占据主空间, 15 通道 matrix 因 journal 截断未显示

**Owner 2026-06-28 18:13 实战修订反馈** (v3 触发, 最终方向):
> "通道不要变，还是LSB和MSB分别15通道，也就是要显示30通道。但之前做的展示有问题。从之前的效果来看，有非0比特位的信息（纵向，比如bit=1(MSB)）。横向头部有RGB、RBG、GRB、GBR、BRG我实在看不明白"

**关键点解析**:
1. **通道不要变** = 15 通道保留 15 通道
2. **LSB 和 MSB 分别 15 通道 = 显示 30 通道** = 横向 15 通道 × 纵向 bit (含 LSB/MSB)
3. **bit=1 (MSB) 有信息** = mid-bit (bit=1~6) 实战有价值, 不能砍
4. **6 perm 看不明白** = 横向表头从 6 个 perm (RGB/RBG/GRB/GBR/BRG/BGR) 改 15 个通道 (含 RG0/R0B/0GB/R00/0G0/00B/R/G/B)

### 1.2 最终方案: 8 bit × 15 channel 矩阵

**横向表头 (15 通道)**: per Owner 列表 (15 个, 含 zero-padded + single channel)
**纵向表头 (8 bit)**: bit=0 LSB / bit=1 / bit=2 / ... / bit=7 MSB (保留所有 mid-bit)
**总 cells**: 8 × 15 = **120 cells**
**每 cell**: 16 byte ASCII preview + `<==` 关键字标记

**实战价值**:
- bit=0 LSB 行: 实战 95% LSB 隐写命中区
- bit=7 MSB 行: 罕见但存在 (zsteg `b8` 模式)
- bit=1~6 行: mid-bit 实战有价值 (Owner 反馈 bit=1 行有信息)
- 横向 15 通道含 RG0/0GB (N=NP 衍生) + R/G/B (单通道) + 6 full perm + 4 其他 zero-padded, 实战全场景覆盖

### 1.3 决策历史 (vs v1 KEEP+ADD / v2 7 通道)

| 版本 | 决策 | 实战结果 |
|---|---|---|
| v1 (17:25) | KEEP+ADD 8×6 + 15ch 双 matrix | journal 太长 3KB, 15ch 因截断不显示 (Owner 左图) ❌ |
| v2 (18:06) | REPLACE 8×6 + 精简 7 通道 | 通道砍太多, 实战可能漏命中 ⚠️ |
| **v3 (18:13, 最终)** | **8 bit × 15 channel** | **通道命名直白 + 保留 mid-bit + 全 120 cells** ✅ |

## 3. 详细设计 (v3 最终)

### 3.1 `_15_CHANNELS` (15 通道定义, 作为 source of truth)

```python
_15_CHANNELS: list[tuple[str, list[str]]] = [
    # 6 full perm (实战 95%)
    ("RGB", ["R", "G", "B"]),
    ("RBG", ["R", "B", "G"]),
    ("GRB", ["G", "R", "B"]),
    ("GBR", ["G", "B", "R"]),
    ("BRG", ["B", "R", "G"]),
    ("BGR", ["B", "G", "R"]),
    # 6 zero-padded (positional zero 跟 随波逐流 一致, RG0 ≠ RG 当 G bit ≠ 0)
    ("RG0", ["R", "G", "0"]),
    ("R0B", ["R", "0", "B"]),
    ("0GB", ["0", "G", "B"]),
    ("R00", ["R", "0", "0"]),
    ("0G0", ["0", "G", "0"]),
    ("00B", ["0", "0", "B"]),
    # 3 single channel (N=NP 模式金标准)
    ("R", ["R"]),
    ("G", ["G"]),
    ("B", ["B"]),
]
```

### 3.2 `_build_bit_plane_preview_matrix` (8 bit × 15 channel)

```python
def _build_bit_plane_preview_matrix(
    img_array, n_bytes=16, scan_order="row", byte_bit_order="msb"
):
    """8 bit × 15 channel preview 矩阵 (per v3 实战修订).
    
    横向表头: 15 通道 (per _15_CHANNELS)
    纵向表头: bit (b0=LSB ~ b7=MSB)
    总 cells: 8 × 15 = 120
    """
    matrix = []
    for bit in range(8):
        for label, channels in _15_CHANNELS:
            byte_stream = _extract_lsb_byte_stream_zero_aware(
                img_array, channels=channels, bit=bit,
                scan_order=scan_order, byte_bit_order=byte_bit_order,
            )
            preview = _ascii_preview(byte_stream, n_bytes=n_bytes)
            has_kw = any(kw in byte_stream[:n_bytes] for kw in _PREVIEW_KEYWORDS)
            matrix.append((bit, label, preview, has_kw))
    return matrix
```

### 3.3 `_format_matrix_for_journal` (15 列渲染)

```python
def _format_matrix_for_journal(matrix, col_width=16):
    """把 8 bit × 15 channel 矩阵渲染成 journal 字符串."""
    if not matrix:
        return ""
    channel_names = [label for label, _ in _15_CHANNELS]
    header_cells = [f"{p:>{col_width}s}" for p in channel_names]
    header = " " * 18 + "".join(header_cells)

    lines = [header]
    for bit in range(8):
        if bit == 0:
            label_text = "bit=0 (LSB):"
        elif bit == 7:
            label_text = "bit=7 (MSB):"
        else:
            label_text = f"bit={bit}:      "
        row_cells = []
        for ch_name in channel_names:
            entry = next((e for e in matrix if e[0] == bit and e[1] == ch_name), None)
            if entry is None:
                row_cells.append(" " * col_width)
                continue
            _, _, preview, has_kw = entry
            cell = preview.ljust(col_width)
            if has_kw:
                cell = cell.rstrip() + " <=="
                cell = cell.ljust(col_width + 4)
            row_cells.append(cell)
        lines.append(f"{label_text:18s}" + "".join(row_cells))
    return "\n".join(lines)
```
