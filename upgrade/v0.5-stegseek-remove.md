## 1. 背景

### 1.1 stegseek 现状（per `v0.5-philosophy-rethink`）

- `stegseek` 0.6: Linux/macOS 原生 Win 不可用
- automisc macOS 时代为绕开 steghide 编译不带 JPEG 问题，引入 stegseek 作为"现代 fork"
- automisc 端代码已埋 fallback：Win/Linux 实际跑 steghide

### 1.2 为什么现在删（per Owner 2026-06-28 16:03）

- 项目 `AGENTS.md §2.3 v3.3` Windows only，stegseek Win 不可用
- Win 端 `SteghideAdapter` (name="stegseek") 已实际调 steghide — 删 stegseek 不影响 Win 行为
- macOS/Linux 在 v0.5+ 不评估范围
- 留着 stegseek = 误称 + 误导 + 代码冗余

### 1.3 删除后的能力矩阵

| 能力 | 现状 (stegseek/steghide 双轨) | 删后 (统一 steghide) |
|---|---|---|
| **JPEG/BMP/WAV/AU 探测** (auto_run) | Win 走 steghide info / macOS 走 stegseek --crack 空 wordlist | Win 走 steghide info + **空密码 extract 兜底** (CVE-2021-27211) |
| **空密码提取** (auto_run 关键能力) | macOS stegseek 秒级 / Win silent miss (本 spec 修) | Win 端 **steghide extract -p ""** 兜底 (本 spec 加) |
| **字典暴力破解** (GUI 工具栏手动) | stegseek 0.6 14M 密码/1.21s (macOS) / Win 不可用 | **steghide 循环** (~5h/14M, Win + MacOS + Linux 统一) |
| **指定密码提取** (GUI 工具栏) | stegseek/steghide 双路径 | 统一 steghide extract -p <pw> |

**Net 变化**：
- ✅ Win 端 auto_run 多了**空密码 extract 兜底**（之前 silent miss）
- ✅ Win 端 GUI 工具栏多了**字典暴力破解**（之前 stegseek_crack 走不通）
- ⚠️ 失去 macOS 端 stegseek 0.6 14M 密码/1.21s 速度（v0.5-windows-only 已决定 macOS 不在范围）
- ✅ 代码净简化（删 1 个 action 文件 + 1 套测试 + 100+ 引用统一）

## 5. 风险评估

| 风险 | 概率 | 缓解 |
|---|---|---|
| **stehsgtseek_crack action 删除破坏 chain_runner 测试** | 🟢 低 | 同步改 chain_runner 注册 + 测试 |
| **steghide 空密码 extract 写文件污染测试** | 🟡 中 | 临时文件用 `tempfile.mkstemp` 立刻清，per `StegseekCrackAction` 同样模式 |
| **PySide6 + multiprocessing 集成 SIGABRT** (steghide_crack 加速) | 🟡 中 | **不用 multiprocessing** (per Owner 16:00 拍板 Plan B)，单进程串行可接受 |
| **steghide extract -p "" 在某些 JPEG 上不工作** | 🟢 低 | 5s timeout + exit code 非 0 → 视为空密码失败，不写 SP |
| **删 stegseek 影响 macOS 端** | n/a | per v0.5-windows-only macOS 不在范围 |
| **字典爆破 14M rockyou 5h+** | 🟡 中 | 给 GUI 用户清晰提示 (mini wordlist 自带 100 常用, 30s 内完) |

## 6. 决策点

| 决策 | 选项 | 推荐 |
|---|---|---|
| **steghide_crack 多进程加速** | A) 用 (4 worker) / B) 不用 (单进程) | **B** (PySide6 SIGABRT 风险 + 5h 反正慢, 简单稳定优先) |
| **mini wordlist 兜底** | A) 自带 100 常用 / B) 不自带 | **A** (CTF 高频: 123456/password/qwerty/空, 命中率 >50%) |
| **保留 stegseek_crack 链名做兼容** | A) 改名 steghide_crack / B) 保留 | **A** (新名跟新 action 一致, 删干净) |
| **现有 StegseekExtractAction 怎么处理** | A) 删/重建 SteghideExtractAction / B) 文件改名 | **A** (跟 StegseekCrackAction 同处理) |
