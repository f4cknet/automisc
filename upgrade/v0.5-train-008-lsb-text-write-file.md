## 3. 修复方案 (per Owner 决策)

### 3.1 决策

> "**lsb不论是text，png，zip还是什么形式，只要是lsb中隐藏可疑的信息，都应该输出一个文件，输出的目录与输入文件的目录一致**"

= text 通道抽到的 main 也写到 `<stem>__lsb.txt`，跟 file 分支一样在 input 同目录。

### 3.2 实施细节

修改 `core/actions/lsb_extract.py:198-252` text 分支:

```python
if text_entries:
    seen_texts: list[dict[str, Any]] = []
    best_sensitive: dict[str, Any] | None = None

    for entry in text_entries:
        raw = _run_zsteg_extract(file_path, entry["channel"])
        if raw is None:
            text = entry["content"]
        else:
            text = raw.decode("utf-8", errors="replace").rstrip("\x00").strip()
        severity = score_text_severity(text)
        is_sensitive = severity == 5
        seen_texts.append({...})
        if is_sensitive and best_sensitive is None:
            best_sensitive = seen_texts[-1]
            break

    main = best_sensitive or seen_texts[0]
    flag_candidate = main["text"] if main["sensitive_keyword"] else None

    # v0.5-train-008 fix: text 通道 main 也写文件 (per Owner 决策)
    # 命名规则: <stem>__lsb.txt (跟 file 分支一致, output_path_for 同目录)
    main_extracted_path = output_path_for(file_path, suffix=".txt", purpose="lsb")
    main_extracted_path.write_text(main["text"], encoding="utf-8")

    return ActionResult(
        success=True,
        data={
            "lsb_text": {
                "channel": main["channel"],
                "text": main["text"],
                "extracted_path": str(main_extracted_path),  # ← 新字段
                "severity": main["severity"],
                "sensitive_keyword": main["sensitive_keyword"],
                "length": main["length"],
            },
            "lsb_texts_scanned": seen_texts,
            "lsb_text_found": True,
            "extracted_files": [str(main_extracted_path)],  # ← 跟 file 分支 schema 一致
            "flag_candidate": flag_candidate,
        },
        message=(
            f"lsb_extract: 命中 LSB text (channel={main['channel']}, "
            f"severity={main['severity']}, len={main['length']}, "
            f"扫描了 {len(seen_texts)} 个 text 通道)"
            + " [高亮] 命中敏感关键词" * main["sensitive_keyword"]
            + f" → {main_extracted_path}"  # ← 新: 写文件路径
        ),
    )
```

### 3.3 不改 file 分支

file 分支已对（line 254-304），不动。

### 3.4 命名规则

| 分支 | 命名 | 例 |
|---|---|---|
| text main | `<stem>__lsb.txt` | `flag11__lsb.txt` |
| file (zip) | `<stem>__lsb.zip` | `flag11__lsb.zip` |
| file (png) | `<stem>__lsb.png` | `flag11__lsb.png` |

**统一 purpose="lsb"** + **suffix 区分形式**。Owner 一眼就能看出哪个是 text 抽的、哪个是 file 抽的。
