## 1. 需求

**问题**：`core/decoders/pyc_decompiler.py` 跑 `.pyc` 反编译依赖 3 个 Python 包：
- `xdis` — 读 magic + version (Py2.x/Py3.x 路由)
- `uncompyle6` — Py2.x 反编译
- `decompyle3` — Py3.x 反编译

这三个包**不在** `pyproject.toml` 依赖里（per v0.5-pyc-magic-sniffer spec: "❌ 不引入新依赖"，spec 假设 owner v0.5-lsb-byte-stream-extract smoke 时装过）。Owner 重装环境 / 新机器跑 automisc 时就缺这 3 个包，decoder 跑不动。

**实战证据**（per `v0.5-train-019-flag-pyc.md` §2.4）：
```
> automisc decode pyc_decompiler --file flag.pyc
[!] xdis load_module failed: No module named 'xdis'
```

**目标**：把 xdis / uncompyle6 / decompyle3 加到 `extend-tools/install.ps1` 的 `$pip_packages` 数组，让 owner 跑一次 install.ps1 就能装齐依赖。

## 2. 设计

### 2.1 版本锁定

| 包 | 版本 | 原因 |
|---|---|---|
| **xdis** | `6.1.7` | uncompyle6 3.9.3 要 `xdis<6.2.0`, decompyle3 3.9.3 要 `xdis<6.3` — 共同兼容范围 xdis 6.1.x (6.1.1 ~ 6.1.7) |
| **uncompyle6** | `3.9.3` | Py2.x 反编译 (PyPI latest 2026-07) |
| **decompyle3** | `3.9.3` | Py3.x 反编译 (PyPI latest 2026-07) |

**关键决策**：不用 xdis `6.3.0`（PyPI latest）— 跟 uncompyle6 3.9.3 冲突 (`ResolutionImpossible: for help visit https://pip.pypa.io/en/latest/topics/dependency-resolution/...`)。

### 2.2 Python 3.13+ 兼容

**问题**：xdis 6.1.x wheel 标 `Requires-Python>=3.6,<3.13`，但 Owner 用 Python 3.13.6 实际能跑（v0.5-train-019 实战验证）。

**修法**：install.ps1 Stage 2 pypi 分支检测 Python 版本，3.13+ 自动加 `--ignore-requires-python`：

```powershell
$py_ver = & python -c "import sys; print(f'{sys.version_info.major}.{sys.version_info.minor}')" 2>$null
$ignore_py_arg = ""
if ($py_ver -and ([version]$py_ver).Major -ge 3 -and ([version]$py_ver).Minor -ge 13) {
    $ignore_py_arg = "--ignore-requires-python"
    Write-Host "[py3.13+] $($pkg.name) v$($pkg.version) (with --ignore-requires-python) ..." -NoNewline
}
& python -m pip install --proxy $ProxyUrl --quiet $ignore_py_arg "$($pkg.name)==$($pkg.version)"
```

### 2.3 跳过检查

install.ps1 现有逻辑用 `python -c "import pkg; print(pkg.__version__)"` 跳过已装的包。这 3 个包都有 `__version__`，跳过检查正常：
- xdis 6.1.7: `xdis.__version__ = "6.1.7"` ✓
- uncompyle6 3.9.3: `uncompyle6.__version__ = "3.9.3"` ✓
- decompyle3 3.9.3: `decompyle3.__version__ = "3.9.3"` ✓
