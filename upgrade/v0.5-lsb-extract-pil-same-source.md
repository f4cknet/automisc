## 2. 目标

### 2.1 能力 A: 抽公共函数 `_extract_lsb_byte_stream`

**目标**: 抽公共函数 `_extract_lsb_byte_stream(arr: np.ndarray, perm: tuple, scan: str) -> bytes`, 跟 lsb_detect 行为**完全一致** (HWC interleaved + MSB first), 供 `lsb_extract.py` + `lsb_detect.py` 复用。

**位置**: 新建 `core/actions/_lsb_byte_stream.py` (~50 LOC, 纯函数, 无外部依赖)

**算法** (per v0.5-train-011 修复后):
```python
def _extract_lsb_byte_stream(arr: np.ndarray, perm: tuple, scan: str) -> bytes:
    """抽 3 通道 (按 perm 顺序) bit 0 plane, HWC interleaved flatten, 8 bit 拼 byte MSB first."""
    bit0_planes = arr[:, :, list(perm)] & 1
    if scan == "row":
        bits_flat = bit0_planes.flatten()
    else:
        bits_flat = bit0_planes.T.flatten()
    n_bytes = len(bits_flat) // 8
    byte_stream = np.zeros(n_bytes, dtype=np.uint8)
    for bit_pos in range(8):
        byte_stream |= (bits_flat[bit_pos::8][:n_bytes] << (7 - bit_pos)).astype(np.uint8)
    return bytes(byte_stream.astype(np.uint8).tolist())
```

### 3.2 核心算法

**(1) 公共函数** `_lsb_byte_stream.py`:
```python
"""LSB 字节流抽取公共函数 (per v0.5-lsb-extract-pil-same-source spec).

供 LSBDetectAction + LSBExtractAction 复用, 跟 lsb_detect 行为完全一致:
- HWC interleaved flatten (per pixel R/G/B 顺序, 跟 zsteg 一致)
- 8 bit 拼 byte MSB first (跟 v0.5-lsb-byte-stream-extract LSBBytesExtractAction 默认一致)
- 不写文件 (per AGENTS §1 铁律 7)
"""
from __future__ import annotations
import numpy as np


def _extract_lsb_byte_stream(arr: np.ndarray, perm: tuple, scan: str = "row") -> bytes:
    """抽 3 通道 (按 perm 顺序) bit 0 plane, HWC interleaved, 8 bit 拼 byte MSB first.

    Args:
        arr: (H, W, 3) uint8 RGB array (PIL Image.convert('RGB'))
        perm: 3 元素 tuple 通道排列 (e.g. (0, 1, 2) = RGB)
        scan: "row" 行扫描 / "col" 列扫描 (writeup 风格)

    Returns:
        bytes 字节流 (len = H*W*3/8)
    """
    bit0_planes = arr[:, :, list(perm)] & 1
    if scan == "row":
        bits_flat = bit0_planes.flatten()
    else:
        bits_flat = bit0_planes.T.flatten()
    n_bytes = len(bits_flat) // 8
    if n_bytes < 16:
        return b""
    byte_stream = np.zeros(n_bytes, dtype=np.uint8)
    for bit_pos in range(8):
        byte_stream |= (bits_flat[bit_pos::8][:n_bytes] << (7 - bit_pos)).astype(np.uint8)
    return bytes(byte_stream.astype(np.uint8).tolist())
```
