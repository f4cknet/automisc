# v0.5-IO-widget — OutputView 升级成 InputOutputView

> **创建时间**：2026-06-14 08:50
> **状态**：✅ done (commit (待 push))
> **触发**：Owner "主窗口不仅可拖文件, 打印 auto_run 过程, 从设计上看, 这不仅仅是打印窗口, 而是一个 input 窗口"

---

## 1. 设计目标（per Owner 2026-06-14）

中央 widget 不仅是"打印窗口"，而是 **input 窗口**：
1. 工具 stdout 还能打 (append_text / append_suspicious / append_chain_log ...)
2. **支持删除 input 区内容** (Clear 按钮 / 拖选删除 / 键盘 Cmd+A → Delete)
3. **支持文本输入**（如粘贴自己复制的 hex）
4. **Read-only toggle**（默认锁住防误改；切换后可编辑）
5. **Hex → ASCII 按钮**（典型 meihuai 场景：拖文件 → 跑出 hex → 复制 hex → Clear → Paste → 点按钮 → 出 text）

---

## 2. 实施

### 2.1 InputOutputView (新类)

继承 `QWidget` (不再是 QPlainTextEdit)，结构:

```
+--------------------------------------------------+
| [Clear] [Paste] [Read-only ON] [Hex → ASCII]    |  <- 顶 bar
+--------------------------------------------------+
|                                                  |
|  QPlainTextEdit (深色背景, Menlo 11)             |
|  - 工具 stdout                                     |
|  - 用户粘贴的 input                                 |
|                                                  |
+--------------------------------------------------+
```

### 2.2 顶 bar 4 按钮

| 按钮 | 行为 |
|---|---|
| **Clear** | 清空 text_edit + 加 `[cleared]` 标记 |
| **Paste** | 从剪贴板粘 → 自动 clear 旧内容 → 替换模式 (per Owner: "用户可以删除所有内容, 然后把新内容粘贴进去") → 末尾补 newline |
| **Read-only: ON/OFF** | 切换 text_edit 锁定状态；默认 ON 防止误改；切到 OFF 时追加 `[input mode ON]` 提示 |
| **Hex → ASCII** | 把 input 区文本当 hex/binary/base64/base32 → ASCII（用 `core.decoders.base_convert.detect_and_decode`）|

### 2.3 候选行挑选逻辑

```
input = "=== log ===\n[stderr]\nexit_code: 0\n28372c37290a"
                       ↑ 跳过 (log 装饰)    ↑ 跳过 (短)
                                            ↑ 选中 (base 字符)
```

**规则**：
1. 有 `selected text` → 用 selection
2. 否则过每一行：跳 log 行（`[xxx]` / `===` / `---`），找最后"看起来像 base-encoded"的非空行
3. `looks_like_base` = 全是 `[0-9a-zA-Z+/= \n\r\t]` 字符 + 不以 `//` / `#` 开头

### 2.4 历史 API 兼容

旧名 `OutputView = InputOutputView`（alias），所以 `_on_chain_finished` / `_on_runner_finished` 等所有 append_* 调用零修改。

`toPlainText()` / `setPlainText()` / `appendPlainText()` / `setReadOnly()` 全部转给 text_edit，让老 test (e.g. `test_drop_triggers_auto_run`) 不需改。

## 3. 真实题端到端（meihuai.jpg）

```
$ python -c "
from PySide6.QtWidgets import QApplication
import sys
app = QApplication(sys.argv)
from automisc.gui.output_view import InputOutputView
v = InputOutputView()
v.append_text('=== Chain: binwalk ===')
v.append_text('[stderr] noop')
QApplication.clipboard().setText('28372c37290a')   # meihuai.jpg 真实 hex
v.paste_clipboard()                                # 自动 clear + 替换
v.run_hex_to_ascii()
print(v.toPlainText())
"
[Hex → ASCII] detected=hex
  input:  28372c37290a
  output: (7,7)
```

→ 这正是 meihuai.jpg 隐藏 QR 坐标的 hex 编码格式！(7,7) 是 QR 第 7 行第 7 列的模块。
