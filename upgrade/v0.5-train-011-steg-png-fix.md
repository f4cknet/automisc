**根因 1 (HWC interleaved 错位)**:
- 之前 commit `d66177c` 改成 per channel 连续 (`np.concatenate(bits_per_channel)`) 跟 lsb_bytes_extract 行为对齐
- **错误** — 跟 zsteg `b1,rgb,lsb,xy` 实际行为不一致
- zsteg 用 **HWC interleaved** (per pixel R/G/B 顺序) 抽字节流
- steg.png 实战命中 `b1,rgb,lsb,xy` 用的就是 HWC interleaved, 拼出 "Hey I think..."

**根因 2 (text 判定过严)**:
- 旧 `_is_printable_text` 要求"**整段**字节流全 printable ASCII 32-126"
- 实际 steg.png 字节流: 前 ~150 字节 printable ("Hey I think...st3g0_saurus_wr3cks") + 后面几千字节是图本身 LSB 噪声 (非 printable)
- 整段判定 fail → 漏报

**双 bug 叠加** = 0 SP 命中

### 3.3 双 bug 修复方案

**(1) HWC interleaved 修复** (回滚 commit `d66177c` 错误改动):
```python
# 旧 (per channel 连续 — 错):
bits_per_channel = []
for ch_idx in perm:
    bits_per_channel.append(channel_bits.flatten())
bits_flat = np.concatenate(bits_per_channel)

# 新 (HWC interleaved — 对, 跟 zsteg 一致):
bit0_planes = arr[:, :, list(perm)] & 1
if scan == "row":
    bits_flat = bit0_planes.flatten()  # HWC row-major
else:
    bits_flat = bit0_planes.T.flatten()  # HWC col-major
```

**(2) text 判定修复** (per v0.5-train-011 实战):
```python
# 旧 (整段全 printable — 错):
_PRINTABLE_RE = re.compile(rb"^[\x20-\x7e]+$")
def _is_printable_text(byte_stream: bytes) -> bool:
    return bool(_PRINTABLE_RE.match(byte_stream))  # 整段必须全 printable

# 新 (≥ 20 字节连续 printable 段 — 对, 跟 zsteg 启发式):
_MIN_PRINTABLE_RUN = 20  # 最少连续 printable 字节数
_PRINTABLE_WINDOW = 1000  # 检查前 N 字节
def _is_printable_text(byte_stream: bytes) -> bool:
    if not byte_stream or len(byte_stream) < _MIN_PRINTABLE_RUN:
        return False
    max_run = 0
    current_run = 0
    for b in byte_stream[:_PRINTABLE_WINDOW]:
        if 32 <= b <= 126:
            current_run += 1
            if current_run > max_run:
                max_run = current_run
            if max_run >= _MIN_PRINTABLE_RUN:
                return True  # 早退
        else:
            current_run = 0
    return max_run >= _MIN_PRINTABLE_RUN
```
