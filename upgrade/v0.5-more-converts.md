## 3. 实施

### 3.1 `core/decoders/base_convert.py` 加 6 个新转换函数

```python
def convert_bin_to_ascii(text) -> BaseConvertResult:
    """2 进制串 → ASCII text. 例: 0100100001100101 → He"""

def convert_dec_to_bin(text) -> BaseConvertResult:
    """10 进制整数 → 2 进制串. 支持空格分隔多个. 例: 65 → 1000001"""

def convert_bin_to_dec(text) -> BaseConvertResult:
    """2 进制串 → 10 进制整数. 例: 1000001 → 65"""

def convert_dec_to_hex(text) -> BaseConvertResult:
    """10 进制整数 → 16 进制串 (lowercase, 无 0x 前缀). 例: 255 → ff"""

def convert_hex_to_dec(text) -> BaseConvertResult:
    """16 进制串 → 10 进制整数. 自动剥 0x 前缀. 例: ff 0xdeadbeef → 255 3735928559"""

def convert_ascii_to_bin(text) -> BaseConvertResult:
    """ASCII text → 2 进制串 (每字符 8 bit, 空格分隔). 例: He → 01001000 01100101"""
```

**关键设计**：
- 全部 text_only=True (走 input 区, v0.5-cipher-decoders-textfix 一致)
- 全部用同一 `BaseConvertResult` dataclass (input / detected_format / output_text / errors)
- 错误处理统一返回 `errors=str(e)`, 不抛 (跟 cipher_decoders 一致)
- **hex-dec bug 修**: 不能用 `_strip_text` 因为它会把 `"ff 0xdeadbeef"` 剥成 `"ff0xdeadbeef"`. 每个 token 单独剥 0x 前缀.

### 3.2 6 个新 decoder 注册到 registry

```python
MORE_CONVERTS = [
    ("bin-ascii", "💻 Bin → ASCII", convert_bin_to_ascii, "..."),
    ("dec-bin",   "🔟 Dec → Bin",   convert_dec_to_bin,   "..."),
    ("bin-dec",   "💻 Bin → Dec",   convert_bin_to_dec,   "..."),
    ("dec-hex",   "🔟 Dec → Hex",   convert_dec_to_hex,   "..."),
    ("hex-dec",   "🔢 Hex → Dec",   convert_hex_to_dec,   "..."),
    ("ascii-bin", "🔤 ASCII → Bin", convert_ascii_to_bin, "..."),
]
for name, display, fn, desc in MORE_CONVERTS:
    register_decoder(DecoderSpec(
        name=name, display=display,
        category="convert", cli_cmd=f"decode {name}",
        run=_make_text_runner(name, fn), description=desc,
        text_only=True,
    ))
```

## 4. 端到端 smoke

```bash
$ automisc decode bin-ascii --text "0100100001100101"
output_text: He

$ automisc decode dec-bin --text "65 66 67"
output_text: 1000001 1000010 1000011

$ automisc decode bin-dec --text "1000001 1000010"
output_text: 65 66

$ automisc decode dec-hex --text "255 65535"
output_text: ff ffff

$ automisc decode hex-dec --text "ff 0xdeadbeef"
output_text: 255 3735928559   (修了 _strip_text bug)

$ automisc decode ascii-bin --text "Hi"
output_text: 01001000 01101001
```
